# Jobs
- jobs can run asyncronously 
- supports remoting
- parameter -AsJob

## types of job
- remote job
- background job
- scheduled task: windows scheduled task
- passcheduledjob: ps specific job
- runspace: multithreading powershell process. 
- psworkflowjob: 
- cimjob / wmijob
- pseventjob

## usecases
schedule
multitasking / multithreading
long running background
log collection / inventory

## job states
- notstarted
- running
- stopped / stopping
- completed
- atbreakpoint
- blocked
- disconnected
- failed
